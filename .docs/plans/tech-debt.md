# 기술부채 트래커 (Tech Debt Tracker)

> 살아있는 문서. 에이전트가 코드 표류·중복·죽은 코드를 발견하면 여기에 기록하고,
> 엔트로피 관리 루틴이 주기적으로 정리 PR을 만들 때 참조한다.

## 형식
각 항목: `위치` — `문제` — `제안 조치` — `우선순위(high/med/low)`

## 미해결 항목
- (예) `src/legacy/auth.py` — 구 인증 미들웨어, 신규 경로와 중복 — 제거 후 신규로 통합 — med

## 정리됨 (최근)
- (완료된 항목은 여기로 이동 후 주기적으로 비움)
