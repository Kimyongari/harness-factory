# References

> Put **excerpts** of external library docs, design systems, and API specs here.
> Don't copy whole docs — excerpt only what this project needs, to save tokens.

## Recommended format
- `*-llms.txt` — LLM-friendly excerpts of library usage
- For external docs, record the source URL + the date fetched (they go stale over time).

## Example
- (e.g.) `design-system-llms.txt` — excerpt of in-house design system component usage (source: ..., as of 2026-05-27)
