# 외부 참조 (References)

> 외부 라이브러리 문서, 디자인 시스템, API 스펙의 **발췌본**을 둔다.
> 전체 문서를 복사하지 말고, 이 프로젝트에 필요한 부분만 발췌해 토큰을 절약한다.

## 권장 형식
- `*-llms.txt` — LLM이 읽기 좋게 정리한 라이브러리 사용법 발췌
- 외부 문서는 출처 URL + 가져온 날짜를 함께 기록한다(시간이 지나면 오래되므로).

## 예시
- (예) `design-system-llms.txt` — 사내 디자인 시스템 컴포넌트 사용법 발췌 (출처: ..., 2026-05-27 기준)
